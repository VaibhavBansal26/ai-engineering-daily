# AI Engineering Daily 🚀

A public learning-in-the-open repository. Every day I learn, build, or explore one small AI/ML engineering concept and commit it here.

## What's inside

| Folder | Purpose |
|--------|---------|
| `days/` | One entry per day: notes, code, and takeaways |
| `docs/LEARNING_LOG.md` | Running index of everything learned |
| `docs/GUARDRAILS.md` | Safety, quality, and evaluation rules the automation follows |

## Topic areas

AI Engineering · ML Engineering · LLMs & Agents · MLOps · Open Source contributions · AI Tools & Frameworks

## How this works

This repo is maintained by a multi-agent automation system (built on Claude Cowork):

1. A **topic agent** suggests fresh topics daily (deduplicated against everything already covered)
2. A **learning agent** researches the topic, writes notes + working code
3. **Human approval** is required before every commit
4. A **guardrails agent** reviews quality weekly

## Daily entry format

Each `days/YYYY-MM-DD-topic.md` contains: concept explanation, why it matters, a minimal working example, and 3 key takeaways.
